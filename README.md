# Wheel of Fortune

Wheel of Fortune implemented in Java.

## Usage

### Compile

    make

or (JAR):

    make && make jar

### Run

    java wof/gui/WheelOfFortuneFrame

or (JAR):

    java -jar WheelOfFortune.jar
